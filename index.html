Quest 3 - Ballshooter / Mesh Export Tool
Step 1: Use Scene Setup Tool on Quest 3 (Settings -> Physical Space -> Space Setup -> Set Up)
Step 2: Set a Boundary (Settings -> Physical Space -> Boundary)
Step 3: Load Website and Select "Start AR" (https://jasonharron.github.io)
Step 4: Have fun!
Controls: Main triggers = Shoot balls, X = Hide Room, Y = Export Room to JSON (Exported rooms can be sent via e-mail [attach from Download folder] and viewed in the ThreeJS Editor.).
(NOTE: Also works on Quest 1 with Walls and Quest 2/Pro with Walls+Furniture)
